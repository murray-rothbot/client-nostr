## Client Nostr

This is the repository for the client-nostr project, providing a Nostr client application.

## Installation

```bash
npm install
```

## Running

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Building

```bash
# Build the project
npm run build

# Watch for changes and build automatically
npm run watch
```

## Cleaning

```bash
# Remove the dist folder
npm run clean
```

## Environment Variables

Ensure you have a `.env` file with the necessary environment variables. You can use the `.env.example` file as a template.
